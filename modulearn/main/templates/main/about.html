What it does

ModuLearn brings together course pages, module launches, enrollment flows, analytics, and progress tracking. It also keeps legacy integrations available for workflows that still depend on them.

Course delivery

Supports course instances, modules, invite-code enrollment, and role-specific dashboards.

External tools

Handles LTI launches, embedded tools, and existing integration paths used by the project.

Progress tracking

Stores completion state, completion timestamps, outcomes, and timeline events.

Implementation style

Server-rendered Django templates, shared static assets, and a single application shell across the site.

The application is under active refinement. Bug reports and workflow feedback help guide ongoing cleanup and feature work.
